# my_package
1) calculator 모듈 - 수학 연산을 간단하게 수행하는 모듈입니다. 
-제공 기능 : add, multiply

2) text_tools 모듈 - 텍스트를 처리하거나 분석하는 간단한 문자열 도구 모듈입니다.
   - 제공기능 : reverse, count

3) init 모듈
   - 패기지를 import 할 때 바로 사용할 수 있도록 주요 기능들을 묶어주는 초기화 파일입니다.

4) 패키지 구조
   ```
   pl-project-main
   ㄴcalculator.py
   ㄴ__init__
   ㄴtext_tools
   ```
